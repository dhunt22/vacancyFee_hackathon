# Data Download

The large data files (GeoPackage, CSV, GeoJSON, KML) are too big for GitHub. Download them from Google Drive:

**https://drive.google.com/drive/folders/1m9ZR8J3WFtEcdClQoiDdhB3ngse2LyTI?usp=drive_link**

Place downloaded files in this `hackathon_data/` folder alongside the QML style files.

## Files Available

| File | Size | Description |
|------|------|-------------|
| `parcels_simplified.gpkg` | 293 MB | All 486K parcels with geometry + joined attributes (QGIS) |
| `vacant_parcels.geojson` | 30 MB | 28K vacant parcels with geometry (QGIS / web mapping) |
| `vacant_parcels.csv` | 9 MB | 28K vacant parcels, no geometry (spreadsheets / Python) |
| `vacant_parcels.kml` | 25 MB | 28K vacant parcels (Google Earth Pro) |
| `parcels_trimmed.csv` | 167 MB | All 486K parcels, no geometry (Python / pandas) |
